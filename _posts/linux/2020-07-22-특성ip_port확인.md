## 환경

- Ubuntu 18.04

- XShell 6 (무료 버전)

## 포트 확인

```
# 해당 ip에 port가 열려있다면 아무런 메시지가 뜨지 않는다.
echo > /dev/tcp/<ip>/<port>

# 아래의 명령어를 통해 성공적으로 실행됐는지 확인하자.
# 성공 = 0, 실패 = 0이 아닌 숫자
echo $?
```
